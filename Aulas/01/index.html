HTML

HTML (linguaguem de Marcação de Hipertexto) é o bloco de construção mais básico da web. Define o significado e a assinatura do conteúdo web.

Tag ou Marcação

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibiçã em um navegador web.

Um elemento HTML é separado de outro texto em um documento por "Tags". As Tags consistem no nome do elemento entre "<" e ">".
