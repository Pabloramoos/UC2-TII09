Introdução aos Hiperlinks em HTML

Hiperlinks são um dos elementos fundamentais da web, permitindo a navegação entre páginas e recursos. Em HTML, os hiperlinks são criados usando a tag "a", também conhecida como "anchor" (âncora). Essa tag pode vincular o usuário a outras páginas, arquivos, ou até mesmo seções específicas dentro da mesma página.

A tag "a":

A tag "a" é usada para criar hiperlinks. Ela pode incluir varios atributos, mas os mais comuns são href e traget.

Sintaxe Básica de Tag "a":

Texto do Link.

Atributos Comuns da Tag "a":

  • href: Define o destino do link (URL)
  • traget: Especifica onde abrir o link. Valores comuns são _self (padrão) e _blank(nova aba/janela).
  • title: Adiciona um texto explicativo que aparece ao passar o mouse sobre o link.

Navegação Dentro da Mesma Página

Para criar links que navegam para diferentes seções da mesma página, usamos indetificadores (id) e o atributo href com #.
