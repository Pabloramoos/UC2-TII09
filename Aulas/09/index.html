Introdução à Tag img

A tag img em HTML è usada para incorporar imagens em uma pagina web. Ela é um elemento vazio, ou, seja, mão possui uma tag de fechamento e contém apenas atributos que definem o conteudo visual.

Atributos principais:

  • src (source): define o caminho que será exibida. pode ser uma URl externa ou um caminho relativo dentro do servidor.
